# 🎨 AI Virtual Painter

A real-time AI Virtual Painter built with Python, OpenCV and MediaPipe.

 Features
- ✋ Real-time hand tracking
- 🎨 Draw with index finger
- ✌️ Select colors with two fingers
- 🟣 Pink, 🔵 Blue, 🟢 Green colors
- ⬜ Eraser tool

 Libraries Used
- OpenCV
- MediaPipe
- NumPy

 How to Run
```bash
python VirtualPainter.py
```

 Controls
| Gesture | Action |
|---|---|
| ☝️ One finger | Draw |
| ✌️ Two fingers | Select color |
| C key | Clear canvas for img |
| Q key | Quit |
